# Reorientação do produto

O produto passa a ser um Sistema Operacional de Consultoria Empresarial. O CRM é a infraestrutura de execução comercial; a diferenciação está em transformar metodologia em pesquisa, diagnóstico, decisão, plano e acompanhamento.

## Fluxo central

```text
fontes → metodologia → etapas → agentes/workflows → execução → evidências
→ artefatos → recomendações → plano de ação → CRM/KPIs
```

## Princípios

1. Nenhuma recomendação relevante sem fonte, confiança e limitação.
2. Agente sugere por padrão; execução externa exige aprovação configurável.
3. Dados de clientes são separados do conhecimento metodológico.
4. Cada saída precisa ser persistida e versionada.
5. Mocks não podem representar funcionalidade concluída.
