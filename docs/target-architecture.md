# Arquitetura-alvo

## Camadas

- **Entrada:** onboarding, uploads, URLs, transcrições, Knowledge Base e integrações.
- **Domínio:** organizações, projetos, metodologias, etapas, agentes, workflows e artefatos.
- **Orquestração:** execuções idempotentes, dependências, retries, aprovação e retomada.
- **Entrega:** diagnóstico, plano estratégico, funil, tarefas, KPIs e relatório executivo.
- **Operação:** CRM, inbox, automações e marketing attribution.

## Primeiro vertical slice

Cadastro de contexto → briefing existente → criação de projeto → seleção de metodologia → execução registrada → artefato estratégico → recomendação → ação. A execução de IA externa será conectada por provider depois; o contrato persistente já fica pronto agora.
